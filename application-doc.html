mobile software development

I was assigned to develop an application named MyVehicle using an integrated suite of mobile software development tools including Android Studio, Eclipse Integrated Development Environment, and Android Application Program Interface with languages like XHTML, SQL, and Java. The main purpose of the software was to create, save, display, add, delete, update and search car details like driver's name, registration number, departure time, first break, second break and arrival time and send it to the specific user.
